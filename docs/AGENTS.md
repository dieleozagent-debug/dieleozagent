# 📖 AGENTS: Manual Operativo del Agente

## Flujos Maestros de Trabajo
1. **DBCI Execution**: Validación contra SSOT antes de cualquier cambio.
2. **SIT Simulation**: Simulación de impacto transversal N a N+1.
3. **P42 Debugging**: Aislamiento y fix de causa raíz.
4. **Memory Synthesis**: Destilación de aprendizaje tras cada fix exitoso.

## Directrices de Desarrollo
- **HTML**: Siempre Premium, Glassmorphism, Responsive y con inyección dinámica.
- **JS**: Datos desacoplados, uso de `window.` para visibilidad global en el navegador, pureza UTF-8.
- **Git**: Commits descriptivos, sincronización constante con `LFC2` (Vercel) y `agente` (Brain).

## Criterio de "Hecho" (Done)
Un trabajo está terminado solo cuando ha sido validado, sincronizado en Git y documentado en el `walkthrough.md`.
